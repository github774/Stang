FAQ

Questions & Answers

Prism × Swarm is a scenario engine, not a predictor. It doesn't claim to forecast real-world events — instead it generates theoretically-grounded counterfactuals so researchers can explore mechanisms, test interventions, and understand structural sensitivities in belief propagation.

Prism dimensions are dynamically coupled — they update each other during simulation. A high-trust agent who encounters repeated contradictions doesn't just change a single value; the shift in trust cascades into emotional reactivity, conformity, and openness in ways that reflect real psychological research on belief revision and cognitive dissonance.

The web simulator supports up to 10,000 agents in real-time visualization mode. Research API access enables batch simulations at scale — up to 1M agents — with results streamed to your data pipeline. Graph sparsity and topology type are the primary determinants of performance, not just agent count.

Yes. Swarm accepts graph input in GraphML, GEXF, edge-list CSV, and JSON adjacency formats. You can import anonymized network snapshots from X/Twitter, Reddit, or any platform with available data exports. Prism profile initialization can be seeded from survey datasets using our calibration toolkit.

The core simulation engine will be released under MIT license at the hackathon's conclusion. The web interface and research API are currently in preview. Academic researchers can request early API access — we're prioritizing teams at accredited institutions working on misinformation, collective behavior, or computational social science.
